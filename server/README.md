# server

## environment variables

- `POSTGRESQL_URL`: url of the postgresql database.
- `ALGOLIA_APP_ID`: Algolia Application Id
- `ALGOLIA_API_KEY`: Algolia API Key
- `ALGOLIA_STARTERS_INDEX`: The name of the starters index.
